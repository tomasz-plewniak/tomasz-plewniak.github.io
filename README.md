# tomasz-plewniak.github.io

Personal portfolio website for Tomasz Plewniak — Senior Software Engineer
specializing in .NET, C#, and Azure.

## Tech Stack
- Vanilla HTML/CSS/JS (no frameworks)
- Google Fonts (Plus Jakarta Sans, JetBrains Mono)
- Google Analytics (gtag.js)

## Local Development
Open `index.html` in a browser. No build step required.

## Structure
- `index.html` — Single-page portfolio
- `style.css` — All styling
- `og-image.png` — Open Graph social share image (1200x630)
- `sitemap.xml` / `robots.txt` — SEO files

## Deployment
Automatically deployed via GitHub Pages from the `main` branch.
